Bienvenidos a mi proyecto final del curso JS en coderhouse

Este proyecto es para hacer un lista de tareas en donde se puede:
	Crear lista de tareas , agregarle su priorida y categoria.
	Cuando la actividad esta completada se tacha y tambien se puede eliminar
	
Estoy usando estas librerias:
	Bootstrap + Animate + Toastify + SweetAlert
	
El proyecto esta distribuido de la siguiente manera:
	Página principal : index.html
	Pagina secundari : todoPage.html   

Archivos Js:
	Controla el DOM y eventos: domManager.js
	Lógica de tareas:          taskController.js

Archivos json:
	Categorías :  categorias.json
	Prioridades : prioridades.json
